# Campaign Planning Methodology
# OGZ Saudi Instagram — Evidence-Based Planning Framework
# Derived from 636-observation corpus | Updated: 2026-05-25

---

## The Planning Stack

Every campaign brief runs through 4 layers in order. Skip a layer and the output degrades.

```
1. OCCASION LAYER     → Is there a Saudi calendar moment? Use it.
2. FORMAT LAYER       → What format wins for this occasion × sector?
3. PATTERN LAYER      → Which content pattern fits the goal?
4. VOICE LAYER        → What language, tone, length?
```

---

## Layer 1 — Occasion First

**Rule:** 84% of the corpus is evergreen. Every seasonal occasion outperforms evergreen baseline by +15 to +53pp.

|----------|--------------|-------------------|-----------|
| Winter Seasonal | 100% | +53pp | 7 days |

### Planning Calendar Template

```
T−21: Brief confirmed, occasion locked

```

## Layer 2 — Format Selection

**Rule:** Format is the single highest-leverage decision. Choose wrong and even great content underperforms.

| Format | High-Eng Rate | When to use |
|--------|--------------|-------------|
| **Carousel** | **74%** | Product launches, how-tos, multi-image occasions, menu reveals |
| **Reel** | 65% | Behind-the-scenes, occasion campaigns, entertainment collabs |
| **Image** | 54% | Single-hero product, architectural, cultural moments |
| **Video** | 38% | Use only when motion is essential (not default) |

### Format × Occasion Matrix (corpus-derived)

| Occasion | Recommended Format | Why |
|----------|-------------------|-----|
| Founding Day | Video or Digital Graphic | Motion adds patriotic energy |
| Eid Al-Fitr | Image (branded occasion card) | Clean, shareable, emotional |
| Summer Campaign | Image (outdoor/nature setting) | Fresh, lifestyle appeal |
| Ramadan | Carousel or Image | Depth of occasion content |
| National Day | Carousel or Video | Multiple brand moments |
| Evergreen (F&B) | Carousel | Consistent highest performer |
| Evergreen (Beauty) | Image or Reel | Product showcase + tutorial |

## Layer 3 — Pattern Selection

### Use These (100% high-eng in corpus)

| Pattern | When | Notes |
|---------|------|-------|
| `heritage_storytelling_hook` | Any occasion with Saudi roots | Opens with heritage, connects to present |
| `occasion_specific_greeting` | Any Saudi occasion | Dedicated greeting content, not product-pushed |
| `employee_pride_campaign` | Brand building periods | Team/people stories, authentic |
| `limited_time_promo` | Launch, seasonal, campaign | Must have genuine deadline |
| `entertainment_collab` | Growth phase | Co-create with talent, not just tag |
| `parallel_original_bilingual` | Bilingual audience | Dual-language creative, not translation |

### Avoid These (0–20% high-eng)

| Pattern | Eng Rate | Problem |
|---------|----------|---------|
| `discount_price_dominant` | 0% | Price-first destroys brand perception |
| `csr_brand_story` | 20% | Feels inauthentic, ignored by audience |
| `campaign_teaser` | 20% | Audience disengages before reveal |
| `lifestyle_embed` | 11% | Product gets lost in lifestyle noise |
| `gamified_content` | 20% | Does not translate to Saudi F&B/Beauty audience |

## Layer 4 — Voice & Caption

### Caption Rules (from 273 captioned obs)

| Signal | Rule |
|--------|------|
| **Length** | Target 75+ words. Very long captions = 64% high-eng (+10pp vs baseline) |
| **Hashtags** | Use 0–5 max. 0 hashtags = 45% high-eng. 16+ = 12% |
| **Emoji** | Always include. +6pp lift over no-emoji captions |
| **Language** | Arabic first = 51% high-eng. Bilingual = 44%. English-only = 39% |
| **Opener** | Statement-open (43%) outperforms question-open (39%) |

### Tone × Occasion Guide

| Occasion | Register | Tone | Dialect |
|----------|----------|------|---------|
| Ramadan | formal | reverent, warm | MSA blend or Najdi |
| National / Founding Day | formal | proud, patriotic | Najdi |
| Eid | warm_conversational | celebratory | Najdi or Hejazi |
| Evergreen F&B | warm_conversational | welcoming | Najdi |
| Beauty | warm_conversational | aspirational | bilingual |

## Composition & Visual Rules

**Best compositions (corpus-ranked):**

1. `overhead_spread` — 64% high-eng (feasts, launches)
2. `architectural_framing` — 62% (heritage accounts)
3. `graphic_text` — 61% (occasion cards, campaigns)
4. `product_hero_closeup` — 48% (safe, overused)

**Lighting:**
- Dramatic/moody = 68% high-eng → use for heritage, occasion
- Warm practical = 60% → use for hospitality, food service
- Cold studio = 50% → use for beauty, clean product
- Branded bright = 35% → avoid

---

## Timing Rules

- **Best day to post:** Sunday (61% high-eng)
- **Second best:** Friday (47%)
- **Avoid:** Tuesday (38%)
- **Best months:** March, January
- **Weekends vs Weekdays:** 50% vs 43% — always prefer weekend publish for hero posts

## Planning Checklist

Before briefing production, confirm all 8:

```
☐ Occasion locked (or confirmed evergreen)
☐ Format selected from matrix (not default video)
☐ Content pattern chosen (not discount-dominant)
☐ Caption length target set (75+ words)
☐ Arabic as primary language confirmed
☐ Hashtag count capped at 5
☐ Publish day is Sunday or Friday
☐ Compliance pre-check: no left-hand serving, no mixed-gender contact
```
