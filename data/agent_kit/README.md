# Agent Kit data

Universal registry and templates for **any** OGZ agent.

| File | Purpose |
|------|---------|
| `registry.json` | Agent IDs, Mac homes, Drive folders, harvest paths |
| `drive_folder_template.md` | Google Drive tree to create once |
| `mac_archive_template.md` | Local archive tree on Mac |

Scripts: `scripts/agent_kit/`
